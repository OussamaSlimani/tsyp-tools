Email Template

How to Use:

  1. Install the "Send HTML in Gmail" extension.
  2. Open a blank email in Gmail.
  3. Click on the extension icon.
  4. Paste the copied HTML code into the extension and send your email.

To copy the template code:

1. Right-click on the page.

2. Select "View Page Source".

3. Press Ctrl + A to select all, then Ctrl + C to copy.
